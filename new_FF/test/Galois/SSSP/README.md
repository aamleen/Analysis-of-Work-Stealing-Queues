Program calculates the shortest distance between all nodes in an undirected, weighted graph.

## To run Parallel:
```
$ make
$ ./sssp_par -file <graph> -s <source> -d <destination>
```

## To run Sequential:
```
$ make
$ ./sssp_ser -file <graph> -s <source> -d <destination>
```

# Defaults:
s = 0

d = 1
